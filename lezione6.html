üéØ Il Problema: Componenti Non-Lineari

Finora abbiamo usato rette. Ma un DIODO non segue una retta. Segue una curva esponenziale (Equazione di Shockley).

$$ I = I_s \cdot (e^{\frac{V}{n V_t}} - 1) $$

‚ùå Perch√© la retta fallisce

Se provi a fittare una retta su un diodo, otterrai errori enormi. Il modello preveder√† corrente anche quando il diodo √® spento!

üíª La Soluzione Python (Codice da leggere)

In un ambiente reale, useresti questo codice Python per addestrare un modello polinomiale:

Python: Polynomial Regression
from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import make_pipeline

# 1. Creiamo dati del diodo (Esponenziali)
V = np.linspace(0, 0.8, 20)
I = np.exp(V * 10) - 1  # Semplificazione Shockley

# 2. Creiamo un modello Polinomiale di Grado 6
#    Questo permette alla "retta" di curvare
model = make_pipeline(PolynomialFeatures(6), LinearRegression())
model.fit(V, I)

# 3. Il modello ora cerca di "inseguire" l'esponenziale
#    usando un polinomio.
